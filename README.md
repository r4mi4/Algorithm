# Algorithm

Big O Notation and Algorithm Analysis with Python

        
        | Name	       |    Big O     |
        |--------------|--------------|
        | Constant	   |    O(c)      |
        | Linear       |    O(n)      |
        | Quadratic	   |    O(n^2)    |
        | Cubic	       |    O(n^3)    |
        | Exponential  |    O(2^n)    |
        | Logarithmic  |    O(log(n)) |
        | Log Linear   |    O(nlog(n))|
